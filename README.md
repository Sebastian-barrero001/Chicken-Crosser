# Chicken Crosser: C++ & LLM Project

This is a "Crossy Road" style arcade game developed using **C++** and the **RayLib** library. The project was created as part of an assignment to demonstrate effective prompting with Large Language Models (LLMs) to build a functional, polished application.

## 🚀 Overview
The player controls a chicken trying to navigate across busy roads and dangerous rivers. The goal is to travel as far as possible without getting hit by cars or falling into the water.

## 🛠️ Tools Used
* **Language:** C++
* **Graphics Library:** RayLib
* **AI Tool:** [Gemini]

## 📈 Evolution of the Game
The initial code was generated via an "Ultra-Prompt" for a clean OOP structure. I then evolved the game using the following 5 follow-up prompts:

1. **Level Progression:** Added a system where car speed increases as the score goes up.
2. **Visual Polish:** Implemented randomized car colors and a "death" animation.
3. **Sound Effects:** Integrated RayLib's audio module for movement and collision sounds.
4. **Power-ups:** Added a "Seed" item that grants temporary invincibility.
5. **UI/UX:** Created a Main Menu screen and a local High Score saving system.

## 🕹️ How to Run
1. Ensure you have the RayLib library installed.
2. Compile the `main.cpp` file using a C++ compiler (like MinGW or Clang).
3. Run the executable to play!
